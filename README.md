# 💼 AI-Based Job Recommendation System

An intelligent job recommendation engine that matches candidates
to relevant jobs based on their skills using NLP and Machine Learning.
Built on 2000 real-world job postings from a dataset of 84,000+ listings.

## 🚀 Features

- TF-IDF vectorization for text representation
- Cosine Similarity for semantic job matching
- KNN classifier for job category prediction
- Filter by location and job category
- Interactive Streamlit web application
- Built on real job postings data

## 🛠️ Tech Stack

- Python, Pandas, Scikit-learn, Streamlit, Matplotlib

## 📊 Results

- KNN Accuracy: 87.50%
- Macro F1 Score: 0.97
- 2000 real job postings across 11 categories

## ▶️ How to Run

pip install pandas scikit-learn streamlit matplotlib
streamlit run app.py
